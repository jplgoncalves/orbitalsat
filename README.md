# 🛰️ Simulador de Satélite Virtual - MATLAB

Este projeto é um simulador de órbita terrestre desenvolvido em MATLAB. 

## 🚀 Funcionalidades
* **Visualização 3D:** Terra com textura topográfica e iluminação.
* **Dinâmica Orbital:** Simulação de órbita LEO (Low Earth Orbit) com inclinação ajustável.
* **Sensor de Solo:** Visualização do "footprint" (área de cobertura) do satélite em tempo real.

## 🛠️ Como rodar
Basta abrir o ficheiro `orbitalsat.m` no MATLAB (Desktop ou Online) e clicar em **Run**.
