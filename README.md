## 📋 Resumo do Projeto
O **OrbitalSat** é uma ferramenta técnica desenvolvida para simular o comportamento de satélites em **Low Earth Orbit (LEO)**. O projeto foca-se na precisão visual da projeção do *footprint* (área de cobertura) sobre uma Terra 3D com texturas topográficas.

## 🚀 Especificações e Funcionalidades

| Funcionalidade | Descrição Técnica |
| :--- | :--- |
| **Visualização 3D** | Renderização da esfera terrestre com iluminação solar e texturas de alta definição. |
| **Dinâmica Orbital** | Simulação de trajetórias baseada em parâmetros de inclinação e altitude ajustáveis. |
| **Sensor de Solo** | Cálculo geométrico da área de cobertura projetada em tempo real (Footprint). |

---

## 🛠️ Guia de Utilização

### Requisitos
* MATLAB R2021a ou superior.
* Toolbox de Processamento de Imagem (opcional, para melhores texturas).

### Como Rodar
1. **Download:** Baixe os ficheiros deste repositório.
2. **Execução:** Abra o ficheiro `orbitalsat.m` no editor do MATLAB.
3. **Start:** Clique no botão **Run** na aba superior.
